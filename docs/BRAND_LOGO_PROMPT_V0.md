# Continuum Brand Logo Prompt v0

Use this to generate a raster logo or brand mark for Continuum.

## Prompt

Create a premium technology brand mark for **Continuum**, a protocol for AI agent continuity, public memory, governance, and anchoring.

The logo should feel:

- minimal
- high-end
- sovereign
- futuristic without looking generic
- more like protocol infrastructure than like consumer AI

Visual concept:

- a single iconic symbol, not a flowchart
- suggest continuity across interruption
- suggest identity persistence, replayable memory, and institutional gravity
- evoke a public signal or anchor point
- avoid cute mascots, robots, brains, chat bubbles, chains, coins, or obvious blockchain cliches

Design direction:

- dark, elegant, technological
- luminous cyan / electric blue / cold white highlights
- clean geometry with subtle asymmetry
- strong silhouette, easy to remember
- works on a dark background
- should feel like it could sit beside brands such as high-end protocol, infrastructure, or frontier AI companies

Composition:

- centered symbol
- no surrounding scene
- no mockup
- no extra UI
- no slogans
- transparent or dark background

Typography:

- symbol only, no wordmark

Constraints:

- simple enough to be recognizable at small sizes
- not a generic hexagon
- not a meaningless orbit diagram
- not an infinity sign unless heavily abstracted into a more original form
- no watermark
- no extra text

## Short Variant

Minimal premium logo mark for **Continuum**, an AI agent continuity and public memory protocol. Dark futuristic aesthetic, luminous cyan and blue highlights, single centered symbol, clean geometry, strong silhouette, symbol only, no text, no blockchain cliches, no mascot, no flowchart, no mockup.
